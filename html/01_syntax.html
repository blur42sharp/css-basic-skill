CSS 기본 문법

CSS(캐스케이딩 스타일 시트)는 웹 페이지의 스타일을 담당하는 언어이다.
CSS는 웹 페이지의 다양한 요소에 색상, 크기, 폰트, 레이아웃, 여백 등을 지정하여 페이지의 외형을 조정할 수 있게 해준다.
이를 통해 웹 페이지의 시각적 표현을 세밀하게 제어할 수 있다.

CSS를 적용하는 방법은 크게 세 가지가 있다. 각각의 방법은 상황에 따라 선택해서 사용할 수 있다.

1. 인라인 스타일 (Inline Style)
- HTML 태그 내에서 직접 style 속성을 사용하여 스타일을 지정하는 방법이다.
- 예를 들어, <h1 style="color: blue; font-size: 12px;">CSS 기본 문법</h1>과 같이 style 속성을 통해 태그 안에 스타일을 직접 적용할 수 있다.
- 인라인 스타일은 특정 요소에 빠르게 스타일을 적용할 때 유용하나, 여러 요소에 동일한 스타일을 적용하려면 각 태그마다 스타일을 추가해야 하므로 비효율적이다.
- 또한 HTML과 CSS 코드가 혼합되기 때문에 유지보수와 관리가 어렵고, 코드의 가독성이 떨어질 수 있다.
- 복잡한 프로젝트에서는 인라인 스타일 대신 내부 스타일이나 외부 스타일시트를 사용하는 것이 더 효율적이다.

2. 내부 스타일 (Internal Style)
- HTML 문서 내 <style> 태그를 사용하여 스타일을 설정하는 방법이다.
- <style> 태그 안에 작성된 CSS 규칙은 해당 HTML 문서의 여러 요소에 일관되게 적용될 수 있다.
- 이 방법은 단일 페이지에서 일관된 스타일을 설정할 때 편리하며, 인라인 스타일보다 코드 관리가 용이하다.
- 그러나 여러 HTML 파일에 동일한 스타일을 적용할 경우, 각 파일마다 <style> 태그를 복사해 넣어야 하므로 비효율적이다.
- 웹사이트의 여러 페이지에서 동일한 디자인을 유지하려면 외부 스타일시트를 사용하는 것이 더 효율적이다.

3. 외부 스타일시트 (External Style)
- 스타일을 별도의 CSS 파일로 작성하고, HTML 문서에서 그 파일을 연결하여 사용하는 방법이다.
- 예를 들어, <link rel="stylesheet" href="style.css">와 같이 link 태그를 통해 외부 CSS 파일을 HTML에 포함시킨다.
- 외부 스타일시트를 사용하면 여러 HTML 파일에서 동일한 스타일을 재사용할 수 있어 매우 효율적이며, 코드 유지보수가 용이하다.
- 또한, CSS 파일만 수정해도 연결된 모든 HTML 페이지에 스타일 변경이 반영되므로 관리가 간편하다.
- 이 방법은 HTML과 CSS를 분리하여 웹 표준을 준수할 수 있으며, 코드의 재사용성 및 관리 편의성을 높인다.

외부 스타일시트를 사용하는 방법은 HTML 문서 내에서 스타일을 작성하는 것보다 관리가 용이하다.
여러 페이지에서 동일한 스타일을 적용할 수 있기 때문에, 스타일을 변경할 때 한 번만 수정하면 모든 페이지에 적용되는 장점이 있다.
또한, CSS와 HTML을 분리함으로써 코드가 더 깔끔하고 유지보수가 쉬워진다.
