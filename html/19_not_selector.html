이제 부정 선택자를 사용하여 h1을 제외한 다른 요소들의 색상을 green으로 변경해보겠다.
universal selector(*)를 사용하여 이렇게 header * { color: green; } 적용할 수 있지만, 이렇게 하면 모든 요소가 선택되어 h1까지 색상이 변경된다.
만약 h1을 제외하고 나머지 모든 요소의 색상을 바꾸고 싶다면, :not(h1) 부정 선택자를 사용해야 한다.

위 코드에서 header *:not(h1) { color: green; }처럼 작성하면, h1을 제외한 header의 모든 자식 요소들이 선택되어 색상이 green으로 바뀌게 된다. 이렇게 하면 header 내에서 다른 요소들이 추가되어도 항상 h1을 제외한 나머지 요소들에 스타일을 적용할 수 있다.

부정 선택자는 리스트 항목에서 특정 항목을 제외하고 스타일을 적용할 때도 유용하다.
예를 들어, 리스트에서 마지막 항목을 제외하고 나머지 항목들의 배경색을 실버로 변경하려면,
아래와 같이 li:not(last-child)를 사용하면 된다.

li:not(last-child) { background-color: silver; } 이렇게 작성하면 마지막 li 항목을 제외한 나머지 항목들이 선택되어 배경색이 실버로 바뀌게 된다.
이와 같은 방식으로 부정 선택자를 사용하면 특정 항목을 제외한 나머지 항목들을 선택하여 스타일을 지정할 수 있다.
