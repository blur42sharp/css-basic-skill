CSS 단위 사용 예시

CSS에서 사용할 수 있는 단위에는 pixel(px), percent(%), em, rem, vw, vh 등이 있다.
먼저 pixel(px)과 percent(%)의 사용법을 알아보겠다.

단위

px and percent

pixel

px 단위: .px 클래스를 사용하여 요소의 너비를 200px로 설정하고, 배경색을 silver로 지정하였다. 결과적으로 브라우저 화면에 200px 너비와 해당 배경색이 정확히 표시된다.

Child Element

% 단위: .child 클래스를 사용하여 요소의 너비를 50%로 설정하고, 배경색을 silver로 지정하였다. % 단위는 부모 요소의 크기에 비례하여 적용된다.

예를 들어, 부모 요소 .parent가 브라우저 너비만큼 크다면, 자식 요소 .child의 너비는 그 절반인 50%로 설정된다. 브라우저 너비가 1904px일 경우, .child의 너비는 952px이 된다. 브라우저 너비를 변경하면 자식 요소의 너비도 비율에 따라 동적으로 변경된다.

부모 요소의 너비를 고정값으로 설정하면, 자식 요소의 % 단위는 고정된 부모 너비를 기준으로 계산된다. .parent 요소의 너비를 600px로 설정했을 때, 자식 요소 .child의 너비는 부모 너비의 50%인 300px이 된다.

em and rem

em

em: 단위가 적용된 요소의 폰트 크기를 기준으로 배수로 계산된다. 예를 들어, 기본 폰트 크기가 16px일 때 font-size: 1.5em;은 16 * 1.5 = 24px로 설정된다.

section title(em 적용)

이곳의 폰트 크기는 부모 요소의 폰트 크기를 기준으로 1.5em이 적용되었다. consectetur 텍스트는 부모 크기에서 다시 1.5em이 곱해져 폰트 크기가 더 커진다.

rem: 문서의 기본 폰트 크기(일반적으로 16px)를 기준으로 배수로 계산된다. 예를 들어, font-size: 1.5rem;은 16 * 1.5 = 24px로 설정된다.

Aside title

:root의 기본값을 변경하면 rem을 사용한 모든 폰트 크기가 일괄적으로 조정된다. 예를 들어, :root { font-size: 12px; }로 설정하면 1.5rem은 12 * 1.5 = 18px이 된다.

애초에 18px, 24px, 27px의 크기를 원하는 경우, style에 고정된 px 값을 설정할 수 있다. 하지만 작업이 완료된 웹사이트를 전체적으로 키우거나 줄이고 싶을 때, px 단위를 사용하면 전체적인 크기를 조절할 방법이 없다. 다시 모든 요소를 수정해야 하기 때문이다. 반면, em이나 rem 단위를 사용하면 기준이 되는 body의 폰트 크기나 :root의 폰트 크기만 변경하여 전체적으로 웹사이트의 크기를 조절할 수 있다. 이로 인해 em과 rem 단위를 사용하는 경우가 많다.
