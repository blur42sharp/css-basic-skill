CSS 단위 사용 예시

CSS에서 사용할 수 있는 단위에는 pixel(px), percent(%), em, rem, vw, vh 등이 있다.
먼저 pixel(px)과 percent(%)의 사용법을 알아보겠다.

단위

px

pixel

px 단위: .px 클래스를 사용하여 요소의 너비를 200px로 설정하고, 배경색을 silver로 지정하였다. 결과적으로 브라우저 화면에 200px 너비와 해당 배경색이 정확히 표시된다.

Child Element

% 단위: .child 클래스를 사용하여 요소의 너비를 50%로 설정하고, 배경색을 silver로 지정하였다. % 단위는 부모 요소의 크기에 비례하여 적용된다.

예를 들어, 부모 요소 .parent가 브라우저 너비만큼 크다면, 자식 요소 .child의 너비는 그 절반인 50%로 설정된다. 브라우저 너비가 1904px일 경우, .child의 너비는 952px이 된다. 브라우저 너비를 변경하면 자식 요소의 너비도 비율에 따라 동적으로 변경되는 것을 확인할 수 있다.

만약 부모 요소의 너비를 고정값으로 설정하면, 자식 요소의 % 단위는 고정된 부모 너비를 기준으로 계산된다. .parent 요소의 너비를 600px로 설정했을 때, 자식 요소 .child의 너비는 부모 너비의 50%인 300px이 된다. 브라우저 크기에 관계없이 고정된 비율을 유지한다.
