1. 약관보기 링크와 :target 선택자

상태 선택자(Checked Selector)
약관보기에서 링크에 앞에 #policy라고 되어 있는 것을 확인할 수 있다.
이는 해당 페이지 안에서 id명 policy div쪽 위치로 이동하라는 뜻이다.
브라우저를 확인하면 a태그에 약관보기에 마우스를 올려 클릭해 봤을 때 약관 안내가 나오도록 그 페이지 위치로 이동한다는 뜻이다.
이동하는 것을 확인하려면 약관 안내 부분이 보이지 않게 페이지를 줄여서 확인해보면 된다.

지금 여기에서 a 태그 약관 보기를 눌렀을 때 타겟을 받은 약관 보기의 스타일을 바꿔 보고 싶다고 가정해보자.
그러면 id="policy" 입장에서 타겟을 받았기 때문에 style로 이동해서 타겟을 받으면 이렇게 작성할 수 있다.
#policy:target {} -> :target 이게 링크를 클릭해서 해당 요소가 타겟을 받은 요소를 선택한 것이다.
그럼 그 요소를 선택해서 배경색을 그린으로 글씨를 흰색으로 변경해보자.
#policy:target { background: green; color: #fff; }

브라우저 상에 주소 표시줄에 보면 뒤에 #policy가 있으면 해당 링크로 이동했다는 뜻이다.
만약 해당 부분이 있다면 지우고 다시 확인해보자 이러면 포커스가 타겟이 안 먹게 된다.
이 상태로 약관보기를 클릭하면 이 요소인 id="policy"가 타겟을 받았기 때문에 설정한 대로 초록 배경에 흰색 글씨로 색상이 변경되는 것을 확인할 수 있다.
이렇게 해서 상태 선택자 중에 타겟을 확인해보았다.

2. Radio 버튼과 :checked 선택자

그다음으로 약관 보기 밑에 보면 Yes와 No 중에 하나를 고르는 게 있는데 지금 Yes를 클릭했을 때 form에서 배웠던 것처럼 label의 for와 input의 id가 매치되어 있기 때문에 yes를 클릭하면 input이 체크가 되면서 활성화 된다.
반대로 No를 클릭하면 No가 체크가 된다.
지금 여기에서 만약 체크가 되었을 때 체크된 요소의 스타일을 바꾸고 싶다면?
해당 요소 라벨을 클릭할 때 누가 체크가 되냐면 input이 체크가 된다.
그러면 체크가 된 input부터 먼저 선택을 한다면 style에서 input:checked를 사용한다.
그럼 체크된 input을 선택한 것이다.

하지만 지금 이걸 변경하는 것이 아닌 이게 지금 선택자 중에 radio 버튼이나 또는 checkbox가 클릭이 된 것이다.
활성화가 됐을 때 선택하는 게 :checked 이다.
input:checked
여기에 따라오는 label을 선택하고 해당 요소의 스타일을 바꿀 수 있게 된다.
input:checked + label {} 이라고 작성을 하고 background 색상을 간단하게 silver로 변경해본다.
이 상태로 브라우저에서 No를 클릭해보면 체크된 것 뒤에 따라오는 label이 silver 배경으로 바뀌는 것을 확인할 수 있다.
Yes 체크 역시 마찬가지로 스타일을 바꿔볼 수 있다.

3. Checkbox와 :checked 선택자

마찬가지로 동의 checkbox 부분도 체크가 되었을 때 스타일을 바꾸고 싶다면 해당 요소도 선택해서 바꿀 수 있다.
이것도 마찬가지로 클릭 했을 때 input이니까 체크가 되었을 때 해당 요소의 뒤에 따라오는 라벨의 스타일을 바꿨다.

4. :disabled 선택자

그리고 추가적으로 전송 버튼이 지금 disabled 되어 있어서 클릭이 안되게 설정이 되어 있는데 저 클릭이 안되는 disabled라는 요소를 선택해서 스타일을 바꾸고 싶다면 그럴 때 사용할 수 있는게 :disabled이다.
:disabled라고 하면 지금처럼 disabled 비활성화 되어 있는 버튼을 선택할 수 있다.
스타일을 조금 바꿔보면 기본적으로 브라우저 차원에서 약간 흐리게 표현을 해주고 있지만 이를 수정할 수 있다.
style로 돌아가서 button 중에 disabled 되어 있는 요소를 선택하고 해당 요소의 배경색과 글씨 색을 연한 회색으로 바꿔보겠다.
button:disabled { background: #999; color: #ebebeb; }

이렇게 해서 disabled된 요소의 스타일도 선택해서 변경할 수 있다.
그게 상태 선택자이다.
