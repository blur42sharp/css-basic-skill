태그 선택자는 특정 태그만 선택하여 해당 요소의 스타일을 변경할 수 있다.
예를 들어, h1 태그를 선택하고, color 속성의 값을 blue로 변경하면 제목이 파란색으로 나타난다.
이것이 바로 태그 선택자이다.

이와 같은 방식으로 h2, p 또는 a 태그 선택자는 다양한 선택자들과 함께 사용된다.

하지만 HTML을 작성하다 보면 태그가 많아지기 때문에, 특정 요소를 선택하고 싶을 때는 그 요소에 이름을 지정해 주는 방법이 있다.
이 이름은 일반적으로 id 속성으로 지정된다. id는 각 요소에 대해 고유한 값을 할당할 수 있는 속성이다.

다음 단계에서는 id 선택자에 대해 다뤄보겠다. 08_id_selector.html로 이동해 보겠다.
