CSS는 표현의 언어이다. 우리가 작성한 HTML에서 원하는 요소를 정확하게 선택할 수 있어야 스타일을 제대로 적용할 수 있다.
그 선택 방법 중 첫 번째로 전체 선택자(Universal Selector)를 살펴본다.
전체 선택자는 페이지 내 모든 요소에 스타일을 적용할 수 있는 강력한 도구다.

유니버설 선택자는 * 기호를 사용하여 정의한다.
예를 들어, 아래와 같은 CSS 코드를 사용하면, 페이지 내 모든 요소에 대해 color: green; 스타일을 적용할 수 있다 


        * {
            color: green;
        }

위와 같이 작성하면, h1, h2, p 등의 모든 텍스트 요소들이 green 색상으로 표시된다.
유니버설 선택자는 전체 요소를 대상으로 스타일을 적용하므로, 특정 요소를 제외하려면 더 구체적인 선택자가 필요하다.

유니버설 선택자는 페이지 내 모든 요소를 선택하는 반면, 태그 선택자는 특정 태그에만 스타일을 적용한다.

태그 선택자에 대해서는 07_tag_selector.html에서 더 자세히 확인할 수 있다.

태그 선택자와 유니버설 선택자는 서로 다른 방식으로 적용되며, 다양한 스타일링을 통해 더 세밀한 제어가 가능하다.
유니버설 선택자는 페이지 전체에 공통된 스타일을 적용할 때 유용하게 사용된다.
