Heading 1

HTML 태그를 확인해보면 h1 태그에 id="logo"가 있는 것을 확인할 수 있다. 해당 id="logo"를 선택하여 배경색을 실버로 변경해 보자. id 선택자는 앞에 # (샵) 기호를 사용하여 선택한다. 예를 들어 #logo로 선택하면 id가 "logo"인 요소의 스타일을 지정할 수 있다. 이렇게 하면 id="logo"인 요소의 배경색을 변경할 수 있게 된다. id 선택자는 한 페이지에서 유일한 값을 가져야 하므로, 고유한 식별자가 필요할 때 유용하게 사용된다.

Heading 2

id 선택자는 페이지 내에서 유일해야 한다. 만약 h1과 h2 두 요소 모두에 id="logo"를 사용하면, 브라우저에서는 두 요소 모두 배경색을 적용하지만, 웹 표준 검사 도구에서는 오류가 발생한다. 이는 동일한 id 값이 두 번 이상 사용되면 HTML 문서가 잘못된 구조가 되기 때문이다. 따라서 id는 한 페이지 내에서 한 번만 사용해야 한다.

이처럼 id 선택자는 유일한 요소를 선택할 때 사용된다. 반면 class 선택자는 여러 요소에 동일한 스타일을 적용할 수 있어, 하나 이상의 요소에 같은 스타일을 지정하고 싶을 때 유용하다. 이제 class 선택자에 대해 더 알아보자.

id는 반드시 유일해야 하므로, 동일한 이름을 여러 번 사용하는 것은 HTML 문서 작성에서 오류를 발생시킨다. 반면, class 선택자는 여러 번 사용할 수 있으며, 여러 요소에 동일한 스타일을 적용할 수 있다. 이는 대량의 요소들에 일괄적인 스타일을 적용할 때 매우 유용하다.

그러면, class 선택자에 대한 이해를 돕기 위해 09_class_selector로 이동하여 그 사용 방법을 더 자세히 확인해보겠다.
