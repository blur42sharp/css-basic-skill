Color

Color Name

CSS에서 색상을 지정하는 가장 간단한 방법은 색상 이름을 사용하는 것이다.
예를 들어, color: navy;와 같이 CSS의 color 속성에 색상 이름을 지정할 수 있다.
여기서는 h2 요소의 글자 색상을 navy로 지정하여 어두운 푸른색으로 표현하였다.

HEX Code

두 번째 색상 지정 방식은 헥사(HEX) 코드이다.
헥사 코드는 16진수로 색상을 표현하는 방법이며, 각 색상은 Red, Green, Blue 세 가지로 나뉘어 있다.
HEX 코드는 #RRGGBB 형식을 가지며, 각각 두 자리 숫자가 RGB의 세 가지 색상 값을 나타낸다.
예를 들어, #FF0000은 빨간색이고, 파란색은 #0000FF이다.
두 자리 숫자가 같은 경우에는 #0000FF를 간단히 #00F로 줄여 쓸 수 있다.
이 예제에서는 h3 요소에 파란색 #0000FF를 설정하였다.

RGB Color

세 번째 방식은 RGB 함수를 이용하여 색상을 지정하는 것이다.
RGB는 각각의 색상을 0에서 255 사이의 숫자로 표현하며,
각 값은 Red, Green, Blue 순으로 입력된다.
예를 들어, 파란색은 rgb(0, 0, 255)로 나타낼 수 있다.
이 예제에서는 h4 요소의 글자 색상을 rgb(0, 0, 255)로 설정하였다.

RGBA Color

RGBA는 RGB 값에 알파(Alpha) 값을 추가하여 색상의 투명도를 조절할 수 있다.
알파 값은 0(완전 투명)에서 1(불투명) 사이의 값으로 설정하며, 예를 들어 50%의 투명도를 적용하려면 0.5를 입력한다.
이 예제에서 h5 요소의 배경색을 50% 투명한 파란색 rgba(0, 0, 255, 0.5)로 설정하였다.
이러한 방식으로 특정 색상에 투명 효과를 추가하여 여러 요소 위에서 겹쳐 보이는 효과를 줄 수 있다.

HSL Color

HSL은 색상을 Hue(색조), Saturation(채도), Lightness(명도)의 세 가지 값으로 지정하는 방식이다.
Hue는 0~360도 색상환을 기준으로 색상을 설정하며, 240도는 파란색을 나타낸다.
Saturation은 0~100%로 색의 선명도를 조절하고, Lightness는 0~100%로 밝기를 조정한다.
예를 들어, 파란색은 hsl(240, 100%, 50%)로 나타낸다.
HSL에도 알파 값(a)을 추가할 수 있으며, hsla(240, 100%, 50%, 0.5)와 같이 사용하여 투명도 50%를 적용할 수 있다.
이 예제에서는 h6 요소의 배경색을 50% 투명한 파란색 hsla(240, 100%, 50%, 0.5)로 설정하고, 글자 색상은 흰색으로 대비를 주었다.
또한 padding: 20px로 여백을 추가하여 요소 내의 글자가 적절하게 배치되도록 하였다.
